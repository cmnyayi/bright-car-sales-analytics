# bright-car-sales-analytics
Bright Car Sales Analytics is an end-to-end data analytics project focused on transforming raw vehicle sales data into actionable business insights. The project analyzes sales performance, customer trends, revenue patterns, and vehicle categories to support data-driven decision-making.

# Problem Statement
The business generates large volumes of sales data but lacks a structured approach for identifying trends, monitoring performance, and supporting strategic decisions.

# Aim
To analyze car sales data and develop meaningful insights, dashboards, and recommendations that improve business performance.

# Objectives
Clean and prepare sales data
Transform data using SQL
Create key business metrics and KPIs
Analyze sales and revenue trends
Develop dashboards and visualizations
Generate actionable business recommendations

# Tools Used
Databricks- Data cleaning, transformation, and analysis
Excel- Initial data exploration and validation
Miro- Project planning and project mapping
Canva- Gantt chart design and final presentation
Power BI / Looker Studio- Dashboard development and visualization

# Methodology
1. Project Planning
The project was planned using Miro to define the project workflow, objectives, deliverables, and analytics process.
Deliverables:
Project Plan
Project Map
Gantt Chart

2. Data Understanding
The dataset was reviewed to understand:
Sales records
Customer information
Vehicle categories
Revenue data
Regional performance

3. Data Cleaning & Preparation
Data quality issues were addressed by:
Removing duplicate records
Handling missing values
Correcting data inconsistencies
Standardizing formats
Validating data quality

4. Data Transformation
Using Databricks, data was transformed to support analysis:
Created calculated fields
Standardized date formats
Generated business metrics
Prepared data for dashboard development

5. Data Analysis
The cleaned dataset was analyzed to identify:
Revenue trends
Sales performance
Customer purchasing behavior
Vehicle category performance
Regional sales performance

6. Dashboard Development
Interactive dashboards were created to visualize:
Revenue performance
Monthly sales trends
Top-performing vehicle categories
Customer insights
KPI summaries

# Key Insights
Revenue is concentrated among a few top-performing vehicle brands.
Sales performance varies across regions and time periods.
Certain vehicle categories contribute significantly to profitability.
Seasonal trends influence customer purchasing behavior.

#Recommendations
Focus on high-performing vehicle categories.
Improve marketing in underperforming regions.
Leverage seasonal sales opportunities.
Monitor KPIs regularly to support business growth.
